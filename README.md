# Strava Ride Tracker

![Preview](/docs/assets/preview.png)
A simple web app to analyze my Strava history and compare against regional bike networks in Metro Vancouver/Greater Victoria.

link: https://markedwardson.com/strava-ride-tracker/

## Development - Requirements:
1. Requires a `.env` file at the project root with VITE_STRAVA_MAPBOX_TOKEN.
2. Create a [Strava API app](https://developers.strava.com/) and download your Strava user credentials (`strava_credentials.json`), and add your `STRAVA_CLIENT_ID` and `STRAVA_CLIENT_SECRET` to `.env`.
